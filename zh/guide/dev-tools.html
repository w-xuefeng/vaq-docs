# 开发工具

你可以使用任意你喜欢的开发工具开发 Vue 项目, 但是 WIFI 调试 则需要一些插件来辅助完成。

APICloud 官方推荐的 IDE：APICloud Studio 2

除 APICloud Studio 2 之外，你可以使用以下 IDE 开发和调式项目：

  • Visual Studio Code + APICloud 插件

  • Sublime text 3 + Sublime APIcloud 插件

  • Atom + Atom APIcloud 插件

  • Eclipse + Eclipse APIcloud 插件

  • WebStorm + WebStorm APIcloud 插件
